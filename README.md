## // SYSTEM OVERVIEW

Lumina is a high-performance personal dashboard designed for peak cognitive function. Built with a "Nike-like" aesthetic philosophy—bold, condensed, and motion-centric—it serves as a digital cockpit for deep work.

### CORE MODULES

-   **CHRONO**: Massive typographic focus timer with haptic feedback.
-   **OPERATIONS**: Minimalist task management for high-priority directives.
-   **AUDIO CORE**: Integrated white/brown/pink noise generator with sleek mixing controls.

## // VISUAL LANGUAGE

The interface uses a custom design system focused on "Athletic Minimalism":

| Token | Value | Role |
| :--- | :--- | :--- |
| **VOLT** | `#DFFF00` | Primary accent, energy, active states |
| **ONYX** | `#050505` | Deep background, infinity effect |
| **CARBON** | `#222222` | Surfaces, inactive elements |
| **TYPO** | `Oswald` | Headlines, impact, urgency |

## // TECH STACK

-   **Framework**: Next.js 14 (App Router)
-   **Styling**: Tailwind CSS + Custom Tokens
-   **Motion**: Framer Motion (Physics-based animations)
-   **Icons**: Lucide React
-   **Audio**: Web Audio API (Real-time synthesis)

## // DEPLOYMENT

```bash
npm install
npm run dev
# SYSTEM READY AT LOCALHOST:3000
```
